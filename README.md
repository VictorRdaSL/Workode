# WorKode 💼

A **WorKode** é um projeto interdisciplinar desenvolvido na **FATEC** para a disciplina de **Projeto Interdisciplinar (PI)**.

O projeto consiste em um **portal de vagas** voltado para pessoas que já possuem uma base de estudos e estão iniciando sua trajetória no mercado de trabalho na área de tecnologia. Seu principal objetivo é facilitar o acesso às primeiras oportunidades profissionais, conectando novos talentos a empresas que valorizam aprendizado, desenvolvimento e potencial.

## 🎯 Objetivos

- Aproximar estudantes e profissionais iniciantes do mercado de tecnologia;
- Facilitar a busca por oportunidades de estágio e vagas júnior;
- Proporcionar uma interface simples, intuitiva e acessível;
- Aplicar conceitos de UX/UI Design e Desenvolvimento Web aprendidos durante o curso.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- PHP (elementos básicos)
- Figma (Prototipagem)
- Pesquisa de UX
- Git e GitHub

## 📚 Desenvolvimento

Durante o desenvolvimento do projeto foram aplicados conhecimentos de:

- Arquitetura da Informação;
- Pesquisa com usuários (UX Research);
- Prototipação de baixa e alta fidelidade no Figma;
- Desenvolvimento de interfaces responsivas;
- Estruturação de páginas utilizando HTML e CSS;
- Implementação de funcionalidades básicas com PHP;
- Versionamento de código utilizando Git e GitHub.

## 👨‍💻 Equipe

Projeto desenvolvido como atividade interdisciplinar da **FATEC**, integrando conhecimentos de Design, Experiência do Usuário e Desenvolvimento Web.
